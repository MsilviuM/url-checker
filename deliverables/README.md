# Deliverables

This folder contains final presentation materials that are useful for handoff.

## Files

- `url-threat-checker-university-demo.pptx`
  - Editable PowerPoint presentation for the university demo.
- `url-threat-checker-deck-contact-sheet.png`
  - Quick visual preview of all slides in the deck.

## Model Artifact Reminder

The PowerPoint and repository do not include the trained `.skops` model file.
For the full live demo, share this file separately:

```text
models/url_classifier.skops
```

## Retraining Data Reminder

If the model must be rebuilt, download the private release asset:

```text
url-threat-checker-training-data.zip
```

The training command is documented in the top-level `README.md`.
